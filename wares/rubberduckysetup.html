rubber ducky: quickstart setup

The Hak5 rubber ducky is a fun tool that plugs into a standard USB port and mimics a keyboard, injecting code at high speed onto a target machine.

Unfortunately, setup and startup be a bit difficult for the unfamiliar.

Below is a brief guide to quickly set up the rubber ducky so you can get started coding.
