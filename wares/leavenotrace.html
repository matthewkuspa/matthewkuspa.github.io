leave no trace: deleting your history entries in terminal

This code disables your history, then deletes the history entry disabling the history. After executing the commands, remember to re-enable the history.

Theoretically, this could be used together with rubber ducky or a similar tool to leave no trace in the history of the target machine.

# first type a memorable command
# now pause history

set +o history

# remove reference to set +o history from history

VARHISTREMOVE=$(history | tail -1 | tr -d ' ' | sed "s/set+ohistory//"); history -d $VARHISTREMOVE; unset VARHISTREMOVE

# now type the commands you want hidden
# now re-enable history

set -o history

# now check history, last command should be the first "memorable command" above

history | tail -2
